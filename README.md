## Development

### 1. Setup

To properly run this project, we should install the dependencies:

  ```bash
  make setup
  ```

### 2. Running tests

We have two kinds of tests implemented: unit tests and behaviour tests. We can run the tests with the following commands:

  ```bash
  make test
  ```
